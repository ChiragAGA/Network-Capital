# Network Capital (Phase 2)

## 🚀 Setup

### 1. Supabase
- Create project
- Run `database.sql`
- Enable Google Auth

### 2. Env
Create `.env.local`

NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...

---

### 3. Run locally
npm install
npm run dev

---

### 4. Deploy
- Push to GitHub
- Import in Vercel

---

### 📱 Add to iPhone
- Open site in Safari
- Share → Add to Home Screen

---

## 🔍 Features

- Search by:
  - Name
  - Company
  - Skills
  - Qualifications
  - Hometown
  - City

- Contact CRUD
- Dashboard basics
